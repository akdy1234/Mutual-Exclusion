# Mutual-Exclusion
Enter folder Nodes, and run program Node1.go in 3 seperate terminals

For each seperate terminal, enter port and id: (8000, 1), (8001, 2) & (8002, 3)

Enter any key as prompted ONLY when all 3 servers have been connected

To request access to the critical section, type "Request"

To release access to the critical section, type "Release"
